Available skills include: `digest`, `hn-digest`, `github-trending`, `deep-research`, `paper-digest`, `heartbeat`, and many more.

To run autoresearch on a specific skill, invoke it with `var=<skill-name>`. For example:
- `var=digest` — evolve the digest skill
- `var=hn-digest` — evolve the HN digest skill
- `var=github-trending` — evolve the GitHub trending skill
